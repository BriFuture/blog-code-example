## 华为云鲲鹏开发者大赛

这是 2019 年参加 “华为云鲲鹏开发者大赛” 时编写的一些代码：

`brifuture` 文件夹里面是我编写的策略。只编写了一天，效果并不好，还有很多策略没有实现，打不过 AI，最好的成绩只有 300 多分。

`server` 文件夹里面是模拟官方的服务器，根据官方的文档说明编写的自定义的服务器，大部分功能都实现了，但仍有小部分的 bug 未解决。

**Note：解题策略和服务器仅供参考。**
